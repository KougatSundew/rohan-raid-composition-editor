# World of Warcraft TBC Raid Composition Editor

A modern web application for planning and optimizing 25-man raid compositions in World of Warcraft: The Burning Crusade. Build your raid roster by selecting class specializations and track raid-wide buff and debuff coverage in real-time.

## Features

- **Interactive Raid Roster Builder**: Add up to 25 raid members by selecting from all TBC class specializations
- **Real-time Buff & Debuff Tracking**: Automatically tracks which buffs and debuffs are covered by your current composition
- **Coverage Analytics**: Visual progress indicators showing your raid's buff and debuff coverage percentage
- **Role Distribution**: See the breakdown of tanks, healers, and DPS in your composition
- **Class Color Coding**: Authentic WoW class colors for easy visual identification
- **Responsive Design**: Works seamlessly on desktop and mobile devices

## Tech Stack

- **Next.js 15** - React framework with App Router
- **TypeScript** - Type-safe development
- **Tailwind CSS** - Utility-first styling
- **shadcn/ui** - Beautiful, accessible UI components
- **Lucide React** - Icon library
